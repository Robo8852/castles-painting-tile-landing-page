## 1. Propósito del Proyecto
Este proyecto ofrece al Cliente un **Producto Mínimo Viable (MVP)** que incluye una página de destino, un formulario de contacto, un código QR y un sistema básico de notificaciones. El objetivo es entregar la versión más simple y funcional de la presencia digital del Cliente.  
Este acuerdo **no** incluye marketing, SEO, tráfico garantizado, **ni gestión de redes sociales**.

---

## 2. Entregables (Alcance MVP)
El paquete de $1,500 incluye **ÚNICAMENTE** lo siguiente:

1. Una (1) página de destino adaptable a celulares y computadoras
2. Un (1) formulario de contacto funcional
3. Notificación automática al correo del Cliente cuando alguien envía el formulario
4. Un (1) código QR que dirige directamente a la página de destino
5. Panel de administración básico para visualizar contactos recibidos
6. Configuración sencilla de branding (colores, logo, estructura básica)
7. Preparación del enlace y código QR para que el Cliente pueda compartirlo en redes sociales (sin gestión de publicaciones)

**Stack tecnológico del proyecto:**
- React 18 + TypeScript
- Vite como herramienta de construcción
- Tailwind CSS para estilos
- Encore.dev para hosting y base de datos
- Resend.com para servicio de emails
- Gestión de dependencias con Bun

**Todo lo que no esté listado aquí queda fuera del alcance.**
